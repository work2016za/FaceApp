【This Is Mukuno - 椋野氏顔認証アプリケーション】

■概要
任意の人物画像をアップロードすると、その写真の人物が椋野さんか否かを判定することができるWebアプリケーションです。

■アクセス
http://172.16.0.191:5000

■開始手順
app.py を起動します。

■必要な環境
Python 3.11.4


■必要なモジュール
以下のモジュールが必要です。バージョンに注意してください。

flask==2.3.2
tensorflow==2.13.0
keras==2.13.1
numpy==1.24.3
pillow==9.5.0
